# Traffic Light Robustness Cascade

Репозиторий содержит экспериментальный программный код, подготовленный в рамках выпускной квалификационной работы магистра на тему исследования методов защиты моделей компьютерного зрения в автономных транспортных средствах.

В работе реализована каскадная подсистема распознавания сигналов светофора в видеопотоке. На первом этапе модель YOLOv8s используется для обнаружения светофора и выделения области интереса. На втором этапе дообученный классификатор ResNet-18 определяет состояние сигнала светофора: красный или зелёный.

Также в проекте реализовано стресс-тестирование классификатора с использованием состязательных атак FGSM, BIM и PGD. Для повышения устойчивости классификатора проверяется локальная защитная предобработка области интереса на основе Gaussian Blur и цветового квантования.
